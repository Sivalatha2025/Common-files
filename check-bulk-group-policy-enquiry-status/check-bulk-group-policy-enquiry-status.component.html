Check Bulk Group Policy Enquiry Status

This API is utilized to check the status of multiple policy inquiries in bulk, allowing users to effectively monitor the progress of their requests and obtain detailed information about each policy's status.

QUERY PARAMETERS
ReferenceNumber
required
string
  • Mandatory and validation Don't allow( <>) only.
  • Reference should be as per the response generated in SaveBulkPolicyEnquiry API.
HEADER PARAMETERS
Authorization
required
string
  • Authorization type is Bearer Token
  • Token need to match with system generated else returns error message as "Security Token Verification Failed."
  • In Case of Token expired return error message as "Security Token Expired".
  • If the requested token doesn't have permission to access API's returns error message as "Do not have permission to access this method".
  • If Api Limit count exceeds then return error message as "Api Limit exceed."
